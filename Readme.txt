# Nama Plugins 	: socialfeed
# Base plugins 	: JQuery
# kegunaan		: Hanya untuk mengambil feed dari beberapa social media [facebook, instagram, vk, google, blogspot, twitter, pinterest, rss, weibo]
# Requirement 	: 
	- JQuery
	- codebird.js
	- moment.min.js
	- moment.locale.js

# Examples 		: Silahkan buka folder examples
# Original Github : https://github.com/pavelk2/social-feed/
# Development Github : https://github.com/Dhonisaputra/socialfeed


HOW TO
1. write data from returned data using template / template_html options
untuk memanggil / menulis data pada template / template_html, anda dapat menggunakan pattern {{=it.blablabla}} blablabla dapat diganti berdasarkan data pada dokumentasi _docs_returned_data.js. Anda tidak dapat memanggil data pada object "raw".
